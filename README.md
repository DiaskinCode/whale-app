# Whale — MedTech Health Tracking App

A mobile health companion that helps patients track medication, monitor mood, and share insights with their doctor — all in one place.

---

## The Problem

Patients managing chronic conditions or complex medication schedules struggle with:
- Forgetting to take medications consistently
- No structured way to track how they feel over time
- Doctors making decisions without real behavioral data between visits
- Scattered health information across apps and notebooks

Whale brings it together into one intelligent tracking system.

---

## Core Features

### 💊 Medication Management
- Comprehensive medicine database — search and add medications instantly
- Daily medication schedule with dose tracking
- Mark doses as taken, skipped, or missed
- Full medication history log

### ⏰ Smart Reminders
- Personalized push notification reminders per medication
- Custom reminder schedules (daily, weekly, as-needed)
- Firebase-powered reliable delivery

### 😊 Mood Tracking
- Daily mood check-ins with contextual notes
- Mood history and pattern visualization
- Correlate mood changes with medication adherence

### 📊 Health Analytics & Doctor Reports
- Automatic analysis of medication adherence patterns
- Mood trend reports generated from tracking data
- Shareable reports for doctor appointments — no more "I think I took it most days"
- Insights that help doctors make better-informed decisions

### 💳 In-App Purchases
- Premium features via native IAP (iOS & Android)
- Subscription management

---

## Tech Stack

**Mobile**
- React Native · Expo · TypeScript
- Amplitude (analytics and user behavior tracking)

**Backend**
- NestJS · REST API

**Infrastructure**
- Firebase Cloud Messaging (push notifications)
- EAS (Expo Application Services) for builds and deployment
